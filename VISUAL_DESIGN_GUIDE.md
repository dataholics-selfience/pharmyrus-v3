## Responsive Breakpoints

```
Mobile (< 768px):
- Stack all cards vertically
- Tabs become accordion
- Modal becomes full-screen
- Hide 3D molecule viewer
- Reduce chart height to 240px

Tablet (768px - 1024px):
- 2 column grid for summary cards
- 2 column grid for confidence tiers
- Horizontal tabs (scrollable)
- Modal max-width: 90vw

Desktop (> 1024px):
- 4 column grid for summary cards
- 6 column grid for confidence tiers
- All tabs visible
- Modal max-width: 5xl (1280px)
```

## Design Principles

1. **Scientific Credibility**
   - Clean, professional layouts
   - No gradients or glass effects
   - Subtle shadows (max shadow-md)
   - Border-radius capped at 8px (rounded-lg)

2. **Data-First**
   - Charts and metrics prominent
   - Text secondary to visualizations
   - Progressive disclosure (summary â†’ details)

3. **Legal Compliance**
   - Always visible disclaimers for predicted data
   - Clear separation of confirmed vs predicted
   - Explicit confidence scores
   - Verification reminders

4. **Performance**
   - Virtualized lists (100+ items smooth)
   - Lazy loaded tabs
   - Memoized chart data
   - No layout shift (CLS < 0.1)
